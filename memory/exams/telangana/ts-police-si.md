## Key Notes
- Conducted by TSLPRB (Telangana State Level Police Recruitment Board)
- Posts: Civil SI, AR SI, TSSP SI, Finger Print SI
- Graduation required
- Age: 18–25 years (relaxation for reserved categories)
- PET/PMT qualifying; marks only from written test
- Medical examination after written test + PET
