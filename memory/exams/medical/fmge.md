# FMGE (Foreign Medical Graduate Examination)

board_id  : MCI
category  : C
exam_id   : MCI-FMGE-{YEAR}-{JUN|DEC}
pattern_id: FMGE-PAT-V2

## Sessions
- JUN → MCI-FMGE-2025-JUN
- DEC → MCI-FMGE-2025-DEC

## Pattern
duration: 150 min + 150 min | total_marks: 300 | neg: 0

### Part A (150 min)
Pre & Para-Clinical: Anatomy, Physiology, Biochemistry, Pathology, Microbiology, Pharmacology, FMT → 150Q, 1 mark each

### Part B (150 min)
Clinical: Medicine, Surgery, OBG, Paediatrics, Orthopaedics, Ophthalmology, ENT, Dermatology, Psychiatry, PSM, Radiology, Anaesthesia → 150Q, 1 mark each

## Subject Codes
BIO

## QID Types
S (mcq_single)

## Key Notes
- For Indian citizens with foreign MBBS degree
- Passing = 50% (150/300)
- Conducted by NBE (National Board of Examinations)
- Must pass to practice medicine in India
