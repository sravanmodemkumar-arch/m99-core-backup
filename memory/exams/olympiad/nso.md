## Key Notes
- Conducted by SOF (Science Olympiad Foundation)
- Classes 1–12 eligible
- No negative marking
- Science topics aligned with respective school class syllabus
- Top achievers receive gold/silver/bronze medals + certificates
- Cash prizes for Rank 1 holders per class per state
- NSO is complementary to IMO (Math) and IEO (English)
