# JEE Advanced

board_id  : NTA (conducted by IITs on rotation)
category  : C (Competitive)
exam_id   : IIT-JEE-ADV-{YEAR}-P{1|2}   e.g. IIT-JEE-ADV-2025-P1
pattern_id: JEE-ADV-P1-PAT-V4 / JEE-ADV-P2-PAT-V4

## Papers
- Paper 1 → IIT-JEE-ADV-2025-P1
- Paper 2 → IIT-JEE-ADV-2025-P2
Both mandatory. 3 hours each.

## Pattern — Paper 1 & Paper 2 (same structure)
duration: 180 min per paper | total_marks: 360 per paper

| Section | Subject | Q | Type | Marks | Neg |
|---|---|---|---|---|---|
| Sec A — Single Correct | PHY / CHEM / MATH | 4 | S | +3 | -1 |
| Sec B — Multiple Correct | PHY / CHEM / MATH | 8 | M | +4 partial | -2 wrong |
| Sec C — Numerical | PHY / CHEM / MATH | 6 | N | +4 | 0 |

Each subject gets all 3 sections → 18 questions per subject × 3 subjects = 54Q per paper

## Partial Marking (Multiple Correct — type M)
- All correct options selected: +4
- Partial correct (subset, no wrong): +1 per correct option
- Any wrong option selected: -2

## Subject Codes
PHY, CHEM, MATH

## QID Types
S (mcq_single), M (mcq_multiple), N (numerical)

## Key Notes
- Eligibility: Top 2.5L from JEE Main
- 2 attempts lifetime
- No calculator; rough paper provided
- Used for IIT + ISM Dhanbad admissions
- Pattern changes every year — always verify current year pattern
