# TS ECET (Diploma → Degree Lateral Entry)

board_id  : JNTUH
category  : C
exam_id   : JNTUH-TS-ECET-{BRANCH}-{YEAR}   e.g. JNTUH-TS-ECET-CIVIL-2025
pattern_id: TS-ECET-PAT-V2

## Pattern
duration: 180 min | total_marks: 200 | neg: 0

| Section | Subject | Q | Marks |
|---|---|---|---|
| Mathematics | MATH | 50 | 1 |
| Physics & Chemistry | PHY, CHEM | 50 | 1 |
| Engineering Subject (branch specific) | varies | 100 | 1 |

## Branch Codes (in exam_id)
CIVIL, MECH, EEE, ECE, CSE, CHEM, MINING, MET, TEXTILE, PHARMA

## Subject Codes
MATH, PHY, CHEM + branch subject code

## QID Types
S (mcq_single)

## Key Notes
- Lateral entry to 2nd year B.Tech for Diploma holders
- Telangana state
- Separate paper per branch
- No negative marking
