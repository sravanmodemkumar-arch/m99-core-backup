## PGT (Post Graduate Teacher) Written Test
exam_id: KVS-PGT-{SUBJ}-{YEAR}
duration: 150 min | total_marks: 180 | neg: 0
General sections (60) + Subject-specific (100) + Pedagogy (20)

## Interview / Demo Lesson
60 marks (for all posts)

## Subject Codes
ENG, LANG, GK, REAS, CS, CDP, SUBJ

## QID Types
S (mcq_single)

## Key Notes
- No negative marking
- Central government school network (Kendriya Vidyalayas)
- CTET mandatory qualifying condition
- Final merit: Written (180) + Interview/Demo (60) = 240
- PRT: Graduation + B.Ed/D.Ed | TGT: Graduation + B.Ed | PGT: PG + B.Ed
- NVS (Navodaya Vidyalaya Samiti) has similar pattern
